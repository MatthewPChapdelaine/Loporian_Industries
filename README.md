# Loporian Industries

A small organizational repository to structure the work of Loporian Industries.

## What this repo contains

This repository is used as a top-level organizer for many sub-areas of work including:

- `Artistic_Work`
- `Communications_Work`
- `Consulting_Work`
- `Innovation_work`
- `Investment_Work`
- `Logistics_Work`
- `Matts_Lair_Work`
- `Philosophy_Work`
- `Psychology-Work`
- `Technology-Work`

Each top-level folder contains focused `Research_Paper_Work` and other project directories.

## Contributing

Contributions are welcome — please open an issue or pull request.
